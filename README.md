# E-Commerce UI Design using Figma

This repository contains a desktop-based e-commerce application UI designed using Figma.
The project focuses on maintaining visual consistency, proper alignment, and responsive layout principles.

## Project Overview

* Tool Used: Figma
* Design Type: Desktop UI
* Total Frames: 10
* Focus: Clean layout, consistent spacing, and usability

## Screens Included

1. Home
2. Category
3. Product Listing
4. Product Details
5. Cart
6. Checkout
7. Login
8. Signup
9. Orders
10. Festive Sale

## Key Features

* Uniform desktop frame size across all screens
* Consistent layout grid system
* Proper alignment and spacing
* Dedicated festive sale screen
* Reusable UI components

## Purpose of the Project

This project was created to practice:

* UI design using Figma
* Frame organization and alignment
* Grid systems and spacing consistency
* Designing an e-commerce application interface
